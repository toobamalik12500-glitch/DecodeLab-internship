
# Data Cleaning & Exploratory Data Analysis (EDA) in Excel

## Project Overview
This repository contains two Excel-based data analysis projects. The first project focuses on cleaning the dataset, while the second project performs Exploratory Data Analysis (EDA) to extract meaningful insights.

---

## Project 1: Data Cleaning

### Objective
Prepare the dataset for analysis by improving data quality.

### Tasks Performed
- Removed duplicate records
- Handled missing values
- Corrected data formatting
- Ensured data consistency
- Prepared the dataset for analysis

---

## Project 2: Exploratory Data Analysis (EDA)

### Objective
Analyze the cleaned dataset using statistical methods and charts.

### Analysis Performed
- Mean
- Median
- Minimum & Maximum Values
- Data Distribution
- Charts and Graphs
- Key Insights

---

##  Tools Used
- Microsoft Excel

---

## Files Included
- EDA Analysis File
- Charts

---

##  Conclusion
The data cleaning process improved the quality of the dataset by removing duplicates and correcting inconsistencies. The EDA project summarized the data using statistical measures and visual charts to identify important patterns and trends.
